# Rentable NFT Smart Contract

## Overview
The **RentableNFT** smart contract allows NFT owners to rent out their NFTs for a set price and period. Users can temporarily access digital assets (e.g., in-game items) through a secure on-chain rental system.

## Features
- **Minting NFTs:** The contract owner can create new NFTs.
- **Listing NFTs for Rent:** NFT owners can set a rental price.
- **Renting NFTs:** Users can rent listed NFTs for a specified period.
- **Automatic Expiry:** NFTs return to the owner when the rental period ends.
- **Withdrawal of Funds:** The contract owner can withdraw earned rental fees.
